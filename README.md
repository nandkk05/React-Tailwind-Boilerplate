## Run Locally

Clone the project

```bash
  git clone https://github.com/nandkk05/React-Tailwind-Boilerplate
```

Go to the project directory

```bash
  cd my-react-tailwind-boilerplate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
